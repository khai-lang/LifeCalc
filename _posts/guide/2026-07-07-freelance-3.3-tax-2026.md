  <div class="fl-content-card">
    <h3>1. 프리랜서 세금 구조 — 3.3%가 전부가 아니다</h3>
    <p>프리랜서(인적용역 사업소득자)는 계약금을 받을 때 <strong>3.3%가 자동으로 원천징수</strong>됩니다. 소득세 3% + 지방소득세 0.3%입니다. 그런데 이건 <strong>임시로 떼는 돈</strong>이지 최종 세금이 아닙니다.</p>
    <ul class="accent-list">
      <li><strong>원천징수(3.3%):</strong> 수입이 생길 때마다 자동 납부 — 계약 상대방이 대신 냄</li>
      <li><strong>종합소득세 신고(5월):</strong> 연간 전체 소득을 합산해 실제 세금 확정 — 직접 신고</li>
      <li><strong>정산 결과:</strong> 실제 세금 &lt; 기납부 3.3% → 환급 / 실제 세금 &gt; 기납부 3.3% → 추납</li>
    </ul>
    <p>연 소득이 낮거나 경비가 많으면 대부분 <strong>환급</strong>을 받게 됩니다.</p>
  </div>

  <div class="fl-content-card">
    <h3>3. 5월 종합소득세 — 왜 신고해야 하나</h3>
    <ul>
      <li>프리랜서는 <strong>매년 5월 1일~31일</strong> 종합소득세를 직접 신고해야 합니다 (홈택스 또는 세무서)</li>
      <li>3.3% 원천징수만 했다고 신고가 끝난 게 아닙니다. 신고를 안 하면 <strong>가산세</strong>가 붙습니다</li>
      <li>연 소득 2,400만원 이하 + 다른 소득 없으면 <strong>단순경비율</strong> 적용으로 계산이 간단합니다</li>
      <li>성실신고확인대상자(수입 업종별 기준 초과)는 6월 30일까지 연장</li>
    </ul>
  </div>

  <div class="fl-content-card">
    <h3>5. 실제 세금 계산 흐름</h3>
    <ul class="accent-list">
      <li>
        <strong>① 사업소득금액 계산</strong><br>
        총수입 - 필요경비(단순경비율 또는 실제 경비) = 사업소득금액
      </li>
      <li>
        <strong>② 소득공제 차감</strong><br>
        사업소득금액 - 인적공제(본인 150만원 + 부양가족) - 연금저축·IRP 공제 등 = 과세표준
      </li>
      <li>
        <strong>③ 세율 적용 → 산출세액</strong><br>
        과세표준에 6~45% 누진세율 적용 → 세액공제 차감 = 결정세액
      </li>
      <li>
        <strong>④ 기납부세액 차감 → 환급 또는 추납</strong><br>
        결정세액 - 이미 낸 3.3% = 환급(+) 또는 추가납부(-)
      </li>
    </ul>
    <p class="example-title">💡 예시) 연 수입 3,000만원, 단순경비율 64.3% (IT직종), 부양가족 1인<br>
    사업소득금액: 3,000만 × (1-0.643) = 1,071만원<br>
    과세표준: 1,071만 - 150만(인적공제) = 921만원<br>
    산출세액: 921만 × 6% = 약 55만원<br>
    기납부 3.3%: 3,000만 × 3.3% = 99만원<br>
    <strong>→ 환급 예상: 약 44만원</strong></p>
  </div>

  <div class="fl-content-card bg-light-variant">
    <h3>6. 환급을 더 키우는 절세 방법</h3>
    <ul>
      <li><strong>연금저축·IRP 납입:</strong> 연 900만원까지 세액공제(16.5% 또는 13.2%). 수입이 4,000만원 이하면 납입액의 16.5%를 돌려받습니다.</li>
      <li><strong>노란우산공제(소기업·소상공인공제):</strong> 연 500만원 소득공제. 사업자 등록 후 가입 가능.</li>
      <li><strong>업무용 경비 증빙 수집:</strong> 장비 구매, 소프트웨어 구독, 통신비 일부, 업무 공간 임차료 등 실제 지출 영수증을 모으면 단순경비율보다 많은 경비를 인정받을 수 있습니다 (기준경비율 방식).</li>
      <li><strong>건강보험 지역가입자 절감:</strong> 소득·재산 기준으로 부과. 소득이 낮은 해에 조정 신청 가능.</li>
    </ul>
  </div>

  <div class="fl-content-card">
    <h3>7. 프리랜서 자주 묻는 질문</h3>
    <ul>
      <li><strong>Q. 3.3%만 내면 세금 신고 안 해도 되나요?</strong><br>
      아닙니다. 3.3%는 임시 납부이고 5월에 반드시 종합소득세를 신고해야 합니다. 미신고 시 무신고 가산세(20%)가 붙습니다.</li>
      <li><strong>Q. 연 수입이 적으면 세금이 없나요?</strong><br>
      단순경비율 적용 후 소득공제를 빼면 과세표준이 0이 되는 경우가 많습니다. 이 경우 결정세액이 0원이고, 기납부 3.3%를 전액 환급받습니다.</li>
      <li><strong>Q. 사업자등록을 해야 하나요?</strong><br>
      법적으로는 수입이 있으면 사업자등록 의무가 있습니다. 그러나 소규모 프리랜서는 인적용역 제공자로 처리되어 사업자등록 없이 3.3% 원천징수만으로 처리되는 경우가 많습니다. 연 수입이 크거나 정기적이라면 사업자등록을 권장합니다.</li>
    </ul>
  </div>
