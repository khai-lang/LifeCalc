  <h3>취득세 계산 전 꼭 확인할 점</h3>
  <p>취득세는 취득가액에 세율을 곱하는 것으로 끝나지 않습니다. 주택수, 조정대상지역 여부,
  전용면적, 생애최초 여부에 따라 실제 부담액이 크게 달라집니다.</p>

  <h3 style="margin-top:22px">① 주택수 판단이 세율을 좌우합니다</h3>
  <p>신규 취득으로 세대가 보유하게 되는 주택수를 기준으로 세율이 정해집니다.
  <strong>주거용으로 사용하는 오피스텔도 주택수에 포함</strong>되니 주의하세요.
  업무용 오피스텔은 주택수에서 제외되지만 취득세율은 4%가 적용됩니다.</p>

  <h3 style="margin-top:22px">② 조정대상지역 여부를 반드시 확인하세요</h3>
  <p>같은 2주택이라도 조정대상지역이면 8%, 비조정지역이면 1~3% 세율이 적용되어
  세액 차이가 매우 큽니다. 현재 조정대상지역 지정 현황은 위택스 또는 국토교통부 공고를 통해 확인하세요.</p>

  <h3 style="margin-top:22px">④ 취득세에 더해지는 부가세금</h3>
  <p><strong>지방교육세</strong>: 기본세율(1~3%) 구간은 세율의 10%, 중과세율(8%·12%) 구간은 0.4%가 일괄 적용됩니다.<br>
  <strong>농어촌특별세</strong>: 전용면적 85㎡ 이하 주택은 비과세되며, 85㎡ 초과 시 기본세율 구간 0.2%, 8% 중과 0.6%, 12% 중과 1.0%가 부과됩니다.</p>

  <h3 style="margin-top:22px">⑤ 생애최초 감면, 놓치지 마세요</h3>

  취득세를 최대 200만원(소형주택은 300만원)까지 전액 감면받을 수 있습니다. 감면 혜택은 2028년 12월 31일까지 적용됩니다.</p>

  <h3>자주 묻는 질문 (FAQ)</h3>

  <details>
    <summary>Q. 취득세는 언제까지 신고·납부해야 하나요?</summary>
    <p>취득일(잔금 지급일과 등기 접수일 중 빠른 날)로부터 60일 이내에 신고·납부해야 합니다.
    상속의 경우에는 상속개시일이 속하는 달의 말일부터 6개월 이내입니다. 기한을 넘기면 가산세가 부과됩니다.</p>
  </details>

  <details>
    <summary>Q. 주거용 오피스텔도 주택수에 포함되나요?</summary>
    <p>실제로 주거용으로 사용되는 오피스텔은 주택수에 포함되어 다주택자 중과세율이 적용될 수 있습니다.
    반면 업무용으로 사용하는 오피스텔은 주택수에서 제외되지만 취득세율 4%가 별도로 적용됩니다.</p>
  </details>

  <details>
    <summary>Q. 일시적 2주택도 중과세율이 적용되나요?</summary>
    <p>기존 주택을 일정 기간 내 처분하는 조건의 일시적 2주택은 중과 대상에서 제외되어
    1주택 기준의 기본세율(1~3%)이 적용됩니다. 처분 기한을 넘기면 차액이 추징될 수 있습니다.</p>
  </details>

  <details>
    <summary>Q. 생애최초 취득세 감면은 누구나 받을 수 있나요?</summary>
    <p>소득 요건이 폐지되어 취득가액 12억원 이하 주택을 본인 거주 목적으로 취득하는
    대한민국 국민이라면 소득과 관계없이 최대 200만원(지방교육세 포함 약 220만원) 한도로 감면받을 수 있습니다.</p>
  </details>

  <details>
    <summary>Q. 농어촌특별세는 모든 주택에 부과되나요?</summary>
    <p>전용면적 85㎡ 이하 국민주택 규모 주택은 농어촌특별세가 비과세됩니다.
    85㎡를 초과하는 주택에만 취득세율 구간별로 0.2%~1.0%가 추가로 부과됩니다.</p>
  </details>

  <details>

<script>
(function(){
  function formatComma(el){
    el.addEventListener('input', function(){
      var raw = el.value.replace(/[^0-9]/g, '');
      el.dataset.raw = raw;
      el.value = raw ? Number(raw).toLocaleString('ko-KR') : '';
    });
  }
  document.querySelectorAll('.at-comma').forEach(formatComma);

  document.querySelectorAll('.at-tab').forEach(function(tab){
    tab.addEventListener('click', function(){
      document.querySelectorAll('.at-tab').forEach(function(t){ t.classList.remove('active'); });
      document.querySelectorAll('.at-panel').forEach(function(p){ p.classList.remove('active'); });
      tab.classList.add('active');
      document.getElementById('panel-' + tab.dataset.tab).classList.add('active');
      document.getElementById('at-result-total').textContent = '—';
      document.getElementById('at-breakdown').style.display = 'none';
    });
  });

  function getRaw(id){
    var el = document.getElementById(id);
    return el && el.dataset.raw ? Number(el.dataset.raw) : 0;
  }

  function won(n){
    return Math.round(n).toLocaleString('ko-KR') + '원';
  }

  function baseProgressiveRate(price){
    if (price <= 600000000) return 0.01;
    if (price <= 900000000) return ((price/100000000)*(2/3) - 3) / 100;
    return 0.03;
  }

  function calcHouse(){
    var price = getRaw('at-house-price');
    if (!price) return null;
    var count = document.getElementById('at-house-count').value;
    var adjusted = document.getElementById('at-house-adjusted').checked;
    var over85 = document.getElementById('at-house-over85').checked;
    var firstTime = document.getElementById('at-house-firsttime').checked;

    var rate, isProgressive = false, isMid = false, isHigh = false;
    if (count === '1'){
      rate = baseProgressiveRate(price); isProgressive = true;
    } else if (count === '2'){
      if (adjusted){ rate = 0.08; isMid = true; } else { rate = baseProgressiveRate(price); isProgressive = true; }
    } else {
      if (adjusted){ rate = 0.12; isHigh = true; } else { rate = 0.08; isMid = true; }
    }

    var tax = price * rate;
    var edu = isProgressive ? price * rate * 0.1 : price * 0.004;
    var rural = 0;
    if (over85){
      if (isProgressive) rural = price * 0.002;
      else if (isMid) rural = price * 0.006;
      else if (isHigh) rural = price * 0.01;
    }

    var reduction = 0, eduReduction = 0;
    if (firstTime && count === '1' && price <= 1200000000){
      reduction = Math.min(tax, 2000000);
      eduReduction = Math.min(edu, 200000);
      tax -= reduction;
      edu -= eduReduction;
    }

    return { tax: tax, edu: edu, rural: rural, reduction: reduction + eduReduction, total: tax + edu + rural };
  }

  function calcLand(){
    var price = getRaw('at-land-price');
    if (!price) return null;
    var type = document.getElementById('at-land-type').value;
    var rate = type === 'farm' ? 0.03 : 0.04;
    var tax = price * rate;
    var edu = price * (rate - 0.02) * 0.2;
    var rural = price * 0.002;
    return { tax: tax, edu: edu, rural: rural, reduction: 0, total: tax + edu + rural };
  }

  function calcGift(){
    var price = getRaw('at-gift-price');
    if (!price) return null;
    var type = document.getElementById('at-gift-type').value;
    var totalRate;
    if (type === 'inherit0') totalRate = 0.0096;
    else if (type === 'inherit1') totalRate = 0.0316;
    else if (type === 'gift-general') totalRate = 0.04;
    else totalRate = 0.12;
    var total = price * totalRate;
    return { tax: total, edu: 0, rural: 0, reduction: 0, total: total, allIn: true };
  }

  document.getElementById('at-calc-btn').addEventListener('click', function(){
    var activeTab = document.querySelector('.at-tab.active').dataset.tab;
    var result;
    if (activeTab === 'house') result = calcHouse();
    else if (activeTab === 'land') result = calcLand();
    else result = calcGift();

    var totalEl = document.getElementById('at-result-total');
    var breakdown = document.getElementById('at-breakdown');

    if (!result){
      totalEl.textContent = '금액을 입력해주세요';
      breakdown.style.display = 'none';
      return;
    }

    totalEl.textContent = won(result.total);
    breakdown.style.display = 'block';
    document.getElementById('at-b-tax').textContent = won(result.tax);
    document.getElementById('at-b-edu').textContent = result.allIn ? '위 세율에 포함' : won(result.edu);
    document.getElementById('at-b-rural').textContent = result.allIn ? '위 세율에 포함' : won(result.rural);

    var reductionRow = document.getElementById('at-b-reduction-row');
    if (result.reduction > 0){
      reductionRow.style.display = 'flex';
      document.getElementById('at-b-reduction').textContent = '-' + won(result.reduction);
    } else {
      reductionRow.style.display = 'none';
    }
  });
})();
</script>
