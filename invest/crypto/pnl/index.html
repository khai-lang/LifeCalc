거래내역 기반 PnL 계산기 FIFO/LIFO · 수수료 포함

아래 표에 거래를 입력하세요. 날짜는 표시용이며 계산은 가격·수량·수수료 기준으로 진행됩니다.

날짜 유형 체결가 수량 수수료(%, 또는 정액) 비고 삭제

입력 가이드

  • 수수료가 비율(%)이면 0.1% → 0.1로 입력합니다.
  • 정액이면 거래당 실제 수수료 금액을 입력합니다.
  • FIFO/LIFO는 실현손익 계산에 영향을 주며, 미실현손익은 남은 재고 기준으로 계산됩니다.
  • 손익분기점은 보유 재고 기준으로 왕복 수수료를 고려해 표시합니다.
